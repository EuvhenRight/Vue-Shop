## Project Overview

My first Project on Vue.js with TypeScript. and Tailwind CSS.

## Features

- **Page Favorites:** Add to Cart, Order, and Checkout.

- **Cart:** Tou can do Order.

- **Search:** You can find your favorite Sneakers.
